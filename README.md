# 早报.exe

早报是一款常驻桌面的信息小工具，用来集中查看古诗、英文诗、成语典故和分类新闻。内容可以定时更新，也可以随时手动刷新。

## 主要功能

- 每日生成古诗、英文诗和成语典故
- 查看国际、财经、科技、教育、科学、娱乐和 AI 新闻
- 将选中的内容摘录到日记，并按日期保存
- 自定义显示内容、字体、字号和窗口透明度
- 支持窗口置顶和系统托盘常驻

## 使用方法

1. 双击 `早报v1.2.exe` 启动软件。
2. 点击窗口上方的“设置”，填写 AI 接口地址（末尾加/v1）、密钥和模型，然后点击“测试连接”。接口为兼容 OpenAI 格式，理论上国内外主流大模型均可用。
3. 在设置中选择需要显示的内容和 RSS 信源，也可以调整刷新时间、日记目录及外观。完成后点击“保存”。
4. 点击窗口下方的“刷新”更新当前内容。
5. 切换到新闻后选择分类，点击新闻标题即可用系统默认浏览器打开原文。
6. 在内容中选中文字，再点击“日记”或右键选择“加入日记”，即可保存摘录。未选中文字时，点击“日记”可直接打开当天的日记。通过日记目录的设置，可以与Obsidian的日记功能联动使用。

## 窗口与退出

软件一直隐藏在系统托盘，按设定时间刷新。单击托盘图标或点击最小化可以重新显示窗口；如需彻底退出，请点击界面的X退出按钮。

窗口上方的图钉按钮可让窗口保持置顶。再次点击即可取消。
